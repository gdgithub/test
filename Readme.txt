************************** Intellisys Challenge Readme *******************************

Este espacio es su repositorio para el concurso Intellisys Challenge. Aquí es donde se deben subir todos los elementos necesarios para las entregas y puesta en funcionamiento de las soluciones. Este respositorio estará conectado con una aplicación en la plataforma Heroku en donde el proyecto será desplegado automáticamete desde el "master branch". La dirección donde podrán comprobar la aplicación desplegada se les dará a conocer a cada uno de los equipos.
En este repositorio es importante que la aplicación estable sea subida en el "master branch" ya que este será el que se tome en cuenta para desplegar en Heroku.

Al finalizar cada etapa se deberá subir lo siguiente:

- Carpeta del proyecto del equipo. Tenga en cuenta que esto constituye su aplicación que será desplegada automáticamente en la plataforma Heroku. Por tanto cuide que se encuentre en una versión estable para su corrección. Para más información ver la Guía del concursante.

- Instalador Apk de la aplicación Android que desarrolle.

- Documento del entregable
